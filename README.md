# Objectif
Analyser les performances de joueurs de football et construire un modèle de Machine Learning capable de prédire leur valeur marchande.

# Données
Dataset FIFA 23 Players (Kaggle), contenant les statistiques techniques et physiques des joueurs.

# Étapes
- Nettoyage et préparation des données
- Analyse exploratoire et visualisations
- Étude des corrélations
- Entraînement d’un modèle Random Forest
- Évaluation des performances

# Outils
Python, pandas, numpy, scikit-learn, matplotlib, seaborn
